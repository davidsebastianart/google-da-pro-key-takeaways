## Key Takeaways (Follow the data lifecycle)

- **Database**: a collection of data stored in a computer system
- **Planning Phase** [[6 Stages to The Data Life Cycle]]
	- In this initial phase, businesses determine the type of data needed, management strategies, responsible personnel, and desired outcomes.
	- For example, an electricity provider may decide to collect data on customer electricity usage and building types.

- **Capture Phase**
	- Data is collected from various sources, including external datasets and internal documents stored in databases.
	- Ensuring data integrity, credibility, and privacy is crucial during this phase.

- **Management Phase**
	- This phase focuses on how data is stored, maintained, and secured, which is essential for data cleansing.
	- Proper management ensures that data remains useful and accessible for analysis.

- **Analysis Phase**
	- Data analysts utilize the data to solve problems and support business goals, such as helping customers save energy.
	- This phase is where analytical skills are applied to derive insights from the data.

- **Archiving Phase**
	- Data that is no longer actively used is archived, making it available for future reference without cluttering active datasets.
	- Archiving helps manage the vast amounts of data generated.

- **Destruction Phase**
	- This final phase involves securely erasing data that is no longer needed, protecting sensitive information.
	- For instance, an electricity provider would use secure software to erase data from hard drives and shred paper files.

## Key Takeaways (The data analysis toolbox)

- **Spreadsheets**

	- Data analysts utilize spreadsheets like Microsoft Excel and Google Sheets to collect, organize, and visualize data.
	- They help in identifying patterns and creating visual representations such as graphs and charts.

- **Databases and Query Languages**

	- Databases store structured data, and SQL programs like MySQL and Microsoft SQL Server are used to manage this data.
	- Query languages enable analysts to extract specific information, making data analysis more efficient.

- **Visualization Tools**

	- Tools like Tableau and Looker are used to create visualizations that simplify complex data for stakeholders.
	- These tools facilitate informed decision-making by transforming data into understandable stories.

- **Choosing the Right Tools**
	
	- The choice of tools varies depending on the phase of the data analysis process.
	- Visualization tools are best for creating complex visualizations, while spreadsheets and databases are suitable for organizing and analyzing data.

- **Comparison of Spreadsheets and Databases**

	- Spreadsheets are accessed through software applications and are ideal for small to medium datasets, allowing manual data entry.
	- Databases use query languages, manage large datasets, and support multiple users with structured data organization.

- **Formula:** A set of instructions used to perform a calculation using the data in a spreadsheet

- **Function:** A preset command that automatically performs a specified process or task using the data in a spreadsheet

-  **The Difference Between [[6 Stages to The Data Life Cycle]] and [[The six phases of data analysis]]**: 
	- While the data analysis process will drive your projects and help you reach your business goals, you must understand the life cycle of your data in order to use that process. To analyze your data well, you need to have a thorough understanding of it. Similarly, you can collect all the data you want, but the data is only useful to you if you have a plan for analyzing it.
	
	- The Plan and Ask phases both involve planning and asking questions, but they tackle different subjects. The Ask phase in the data analysis process focuses on big-picture strategic thinking about business goals. However, the Plan phase focuses on the fundamentals of the project, such as what data you have access to, what data you need, and where you’re going to get it.
